template<unsigned int order, unsigned int dim, typename T>
class Tensor< order, dim, T >

Primary Tensor template class

Template Parameters
orderAn unsigned int defining the order of the tensor (1, 2, 3, or 4).
dimAn unsigned int defing the spatial dimension of the problem.
TThe type that is to be stored in the tensor.
